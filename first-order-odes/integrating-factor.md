# Integrating factor.  

## 4. Example: 
Mole balance on a tank with increasing inlet/outlet flow rate. 
````{example} 

```{image}

\begin{align}
\dot n_A &= \dot n_A,in  -  \dot n_A,out
\dot n_A,in &= C_f,A * \dot q * t
\dot n_A &= C_f,A * \dot q * t. - (n_A \ v) * \dot q * t
\end{align}

Rewrite: 
\begin{align }
\dot n_A + ((\dot q * t)\v)_n_A = C_f,A * \dot q * t
\end{align}
