5.7. Integrating factor#

5.7.1. What do we do when an ODE is not exact?#

Example 1:

(5.86)#\[\begin{align} -y \, dx + x \, dy &= 0 && \text{(given)} \\ P &= -y && Q = x \\ \frac{\partial P}{\partial y} &= -1 && \frac{\partial Q}{\partial x} = 1 \\ & & & \Rightarrow \text{Not exact!} \end{align}\]

When solving ODEs, we take the partial derivatives of P and Q. If the partial derivatives are not equal, the equation is not exact:

5.7.2. Finding the Integrating Factor#

Multiply by a factor F to make the ODE exact. This is called the integrating factor.

\[\begin{split} \begin{align*} F &= \frac{1}{x^2} \\ (-y)\left( \frac{1}{x^2} \right) + x\left( \frac{1}{x^2} \right) &= -\frac{y}{x^2} + \frac{1}{x} = 0 \\ \frac{\partial ( F P)}{\partial y} &= -\frac{1}{x^2} \\ \frac{\partial (F Q)}{\partial x} &= -\frac{1}{x^2} \\ \end{align*} \end{split}\]

5.7.3. How did we find F?#

Using this formula:

\[ \begin{align*} R &= \frac{1}{Q} \left( \frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x} \right) \end{align*} \]

  • If R depends only on x, the integrating factor is: $\( \begin{align*} F(x) &= \exp\left( \int R(x) \, dx \right) \end{align*} \)$

If not, check:

\[ \begin{align*} S &= \frac{1}{P} \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) = -\frac{Q}{P} R \end{align*} \]

  • If S depends only on y, the integrating factor is: $\( \begin{align*} F(y) &= \exp\left( \int S(y) \, dy \right) \end{align*} \)$

If neither R nor S are simple functions of only x or y:

You must guess an integrating factor.

5.7.4. Example 1 (continued)#

R Calculation: $$

\[\begin{align*} R &= \frac{1}{x}(-1 - 1) = \frac{-2}{x} \end{align*}\]
\[Thus: \]
\[\begin{align*} F(x) &= \exp\left( \int \frac{-2}{x} \, dx \right) = \exp( -2 \ln(x)) = \left( e^{\ln(x)} \right)^{-2} = \frac{1}{x^2} \end{align*}\]
\[**S Calculation:** \]
\[\begin{align*} S &= -\frac{Q}{P} \cdot R = -\frac{x}{-y} \cdot \left( \frac{-2}{x} \right) = \frac{-2}{y} \\ \end{align*}\]
\[Thus: \]
\[\begin{align*} F(y) &= \exp\left( \int \frac{-2}{y} \, dy \right) = \frac{1}{y^2} \end{align*}\]
\[\]

\boxed{ \begin{aligned} F(x) &= \frac{1}{x^2} \ F(y) = \frac{1}{y^2} \end{aligned} } $$ These are valid integrating factors.

5.8. Application to Linear First-Order ODEs:#

A linear first-order ODE has the form: $$

\[\begin{align*} y' + p(x) \, y = r(x) \end{align*}\]
\[Using an **integrating factor**, we can show that:\]

\boxed{ \begin{aligned} y(x) &= \frac{1}{F(x)} \left( \int F(x) , r(x) , dx + C \right) \ F(x) &= e^{\int p(x) , dx} \end{aligned}} $$

Example: Mole balance on a tank with increasing inlet/outlet flow rate.

A stream with concentration \(c_{{\rm f},{\rm A}}\) of solute A flows into a tank with initial volume V at a continuously increasing volumetric flow rate \(Q = \dot q t\), where \(\dot q\) is the rate of increase in the flow rate and t is time. A well-mixed stream is withdrawn from the tank at the same rate.

If there are initial \(n_{{\rm A},0}\) moles of the solute, derive an expression for the number of moles in the tank \(n_{\rm A}\) as a function of time.

Formulate overall mole balances based on entering and exit streams.

\[ \dd{}{n_{\rm A}}{t} = \dot{n}_{{\rm A},\rm{in}} - \dot{n}_{{\rm A},\rm{out}} \]

Replace the molar flowrates with the equivalent concentrations and volumetric flow rates

(5.87)#\[\begin{align} \dot{n}_{{\rm A},\rm{in}} &= c_{{\rm f},{\rm A}} Q = c_{{\rm f},{\rm A}} \dot{q}t \\ \dot{n}_{{\rm A},\rm{out}} &= \frac{n_{\rm A}}{V} Q = \frac{n_{\rm A}}{V} \dot{q} \end{align}\]

so the initial value problem is

(5.88)#\[\begin{equation} \dot{n}_{\rm A} = c_{{\rm f},{\rm A}}\dot{q}t - \frac{n_{\rm A}}{V} \dot{q}t, \quad n_{\rm A}(0) = n_{{\rm A},0} \end{equation}\]

Rewrite in standard form

(5.89)#\[\begin{equation} \dot{n}_{A} + \left(\frac{\dot{q}t}{V}\right) n_{\rm A} = c_{{\rm f},{\rm A}} \dot{q}t \end{equation}\]

This is a linear first-order ODE with

(5.90)#\[\begin{equation} p = \frac{\dot{q}t}{V} \qquad r = c_{{\rm f},{\rm A}} \dot{q}t \end{equation}\]

that can be solved using an integrating factor:

(5.91)#\[\begin{equation} F = e^{\int p \d{t}} = \exp\left(\int{\frac{\dot{q}t}{V} \d{t}}\right) = \exp\left(\frac{1}{2}\frac{\dot{q} t^2}{V}\right) \end{equation}\]

Then, evaluate the integral

(5.92)#\[\begin{align} \int F r \d{t} &= \int \exp\left(\frac{1}{2}\frac{\dot{q} t^2}{V}\right) c_{{\rm f},{\rm A}}\dot{q}t \d{t} \\ &= \int{e^{u}c_{{\rm f},{\rm A}}Vdu} \\ &= c_{{\rm f},{\rm A}}V e^u \\ &= c_{{\rm f},{\rm A}}V \exp\left(\frac{1}{2}\frac{\dot{q}t^2}{V}\right) \end{align}\]

using the u substitution:

(5.93)#\[\begin{align} u &= \frac{1}{2}\frac{\dot{q} t^2}{V} \\ \d{u} &= \frac{\dot{q} t}{V} \d{t} \end{align}\]

Last, combine the results:

(5.94)#\[\begin{align} n_{\rm A}(t) &= \frac{1}{F} \left(\int F r \d{t} + c \right) \\ &= c_{{\rm f},{\rm A}} V + c \exp\left(-\frac{1}{2}\frac{\dot{q}t^2}{V}\right) \end{align}\]

Use the inital conditions to evaluate the integration constant:

(5.95)#\[\begin{align} n_{\rm A}(0) &= c_{{\rm f},{\rm A}}V + c = n_{{\rm A},0} \\ c &= n_{{\rm A}, 0} - c_{{\rm f},{\rm A}} V \end{align}\]

The final result is:

(5.96)#\[\begin{equation} n_{\rm A}(t) = c_{{\rm f},{\rm A}}V + (n_{{\rm A},0} - c_{{\rm f},{\rm A}}V) \exp\left(-\frac{1}{2}\frac{\dot{q}t^2}{V}\right) \end{equation}\]

Example: Hormone level (again)

We had

(5.97)#\[\begin{equation} c' + kc = A + B\cos\left(\frac{\pi t}{12}\right), \quad c(0) = c_0 \end{equation}\]

This is a linear first-order ODE with:

(5.98)#\[\begin{equation} p = kc \qquad r = A + B\cos\left(\frac{\pi t}{12}\right) \end{equation}\]

that can be solved using an integrating factor:

(5.99)#\[\begin{equation} F = e^{\int k \d{t}} = e^{kt} \end{equation}\]

Then, evaluate the integral using the table of integrals or integration by parts to evaluate the second integral in the equation

(5.100)#\[\begin{align} \int F r \d{t} &= \int e^{kt} \left[A+B\cos\left(\frac{\pi t}{12}\right)\right]\d{t} \\ &= A\int e^{kt} \d{t} + B \int e^{kt} \cos\left(\frac{\pi t}{12}\right)\d{t} \\ &= \frac{A}{k} e^{kt} + B \frac{e^{kt}}{k^{2}+(\pi/12)^2} \Biggl[k \cos{(\frac{\pi t}{12})} + \frac{\pi}{12}\sin{\left(\frac{\pi t}{12}\right)}\Biggr] \end{align}\]

The general solution is:

(5.101)#\[\begin{align} c(t) &= \frac{1}{F} \left(\int F r \d{t} + c^* \right) \\ &= \frac{A}{k} + \frac{B}{k^{2}+(\pi/12)^2} \Biggl[k \cos\left(\frac{\pi t}{12}\right) + \frac{\pi}{12}\sin{\left(\frac{\pi t}{12}\right)}\Biggr] + c^* e^{-kt} \end{align}\]

Use the inital condition to evaluate the integration constant:

(5.102)#\[\begin{align} c(0) &= \frac{A}{k} + \frac{B}{k^{2}+(\pi/12)^2} k + c^* = c_0 \\ c^* &= c_0 - \frac{A}{k} - \frac{B}{k^{2}+(\pi/12)^2} k \end{align}\]

The final solution is:

(5.103)#\[\begin{align} c(t) &= \frac{A}{k} + \frac{B}{k^{2}+(\pi/12)^2} \Biggl[k \cos\left(\frac{\pi t}{12}\right) + \frac{\pi}{12}\sin{\left(\frac{\pi t}{12}\right)}\Biggr] \\ &+ \Biggl[c_0 - \frac{A}{k} - \frac{Bk}{k^{2}+(\pi/12)^2} \Biggr] e^{-kt} \end{align}\]
